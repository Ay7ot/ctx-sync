How It Works

Three commands. That's it.

1

Track

cd ~/projects/my-app
ctx-sync track

Auto-detects Git state, .env files, Docker services. Prompts for mental context. Everything encrypted.

2

Sync

ctx-sync sync

Pushes encrypted state to your private Git repo. Pulls updates from other machines.

3

Restore

ctx-sync restore my-app

Decrypts env vars, shows mental context, prompts to start Docker services. Ready in seconds.

Security First

Designed with a zero-trust, encrypt-everything approach.

Encrypt Everything

All state files encrypted with Age. Not just secrets — project names, paths, and mental context too. An attacker with your Git repo sees only ciphertext.

No Backend

Git is the only transport. No servers, no accounts, no third-party access to your data. Works completely offline.

No Side-Channel Leaks

Secrets never appear in CLI arguments, shell history, process lists, logs, or temp files. Encryption/decryption happens in memory only.

Command Confirmation

Restored commands (Docker, services) always shown for explicit approval. Prevents remote code execution via compromised repos.
